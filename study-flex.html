Flexbox

1. Introduce

Flexbox는 모던 웹을 위해 제안된 세련된 layout 방식이다.
요소의 사이즈가 불명확하거나 동적으로 변화할 때에도 유연한 레이아웃을 실현할 수 있어 복잡한 레이아웃이라도 적은 코드로 보다 간결하게 표현이 가능하다.

이때 각 자식 요소를 부모 요소 내에서 정렬하기 위해서는 각 자식 요소의 너비를 %로 지정하는 등 번거로운 처리가 필요하다.
자식 요소의 사이즈가 불명확하거나 동적으로 변화할 때에는 더욱 처리가 복잡해진다. grid를 사용할 수 도 있으나 이 또한 새로운 학습이 필요하고 라이브러리를 로드해야하는 번거로움이 존재한다.

Flexbox를 사용하여 위 예제를 부모 요소 내에서 균등하게 수평 정렬해보자. 부모 요소에는
.flex-container{ display: flex; justify-content: space-around; }
를 입력한다.

이와 같이 Flexbox를 사용하면 기존 방식에 비해 매우 간단히 레이아웃을 처리할 수 있다.

Flexbox의 장점을 정리해 보면 아래와 같다.

  • 1줄의 코드 추가로 수평정렬이 가능하다.
  • 요소의 상하좌우 정렬, 순서 변경이 가능하다.
  • 요소가 간격 조절이 가능하다.
  • 서로 다른 height를 갖는 요소의 수평정렬 시, 간단한 상하 중앙 정렬이 가능하다.

비교적 최신 브라우저가 아니면 벤더 프리픽스를 사용하여야 하고 IE계열은 IE8,9의 경우 지원하지 않고 IE10,11의 경우도 일부 지원하므로 주의가 필요하다. IE계열에서 Flexbox를 사용하기 위해서는 flexibility.js를 사용하면 편리하다.

브라우저의 지원 상황은 caniuse를 참고한다.

2. Usage

Flexbox 레이아웃은 flex item이라 불리는 복수의 자식 요소와 이들을 내포하는 flex-container 부모 요소로 구성된다.

flex를 사용하기 위해서 HTML 부모 요소의 display 속성에서 flex를 지정한다.

3. Flexbox container 속성


3.1 flex-direction

flex-direction 속성은 flex 컨테이너의 주축(main axis)방향을 설정한다.(즉, 수평배치냐? 수직배치냐?)


flex-direction: row;
